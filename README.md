# Food court

## Built with

- Typescript
- Nodejs
- Nestjs
- Objectionjs
- Knex
- postgres
- Docker & docker compose
- Github actions
- Aws Beanstalk

## Running the app

```bash
# start docker container for Posgres image
$ docker compose up

# Run DB Migrations
$ npm run migrate

# Run aaplication in development mode
$ npm run start:dev
```
